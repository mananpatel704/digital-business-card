# Smart Digital Card – Business Card Management System

## 🚀 Overview
A QR-based digital business card platform that enables instant access to employee profiles with contactless sharing and centralized management.

## ✨ Key Features
- Unique QR code for each employee
- Instant profile access via scan
- Save contact directly to phone
- One-click WhatsApp chat & direct calling
- Admin dashboard for managing company and employees

## 🛠 Tech Stack
PHP, MySQL, JavaScript, HTML, CSS

## ⚙️ Setup
1. Clone the repository  
2. Import the database (.sql file) into MySQL  
3. Run on localhost (XAMPP/WAMP)
